# ticross
An service to play picross, make turnaments, share results and compare with others.

The center is a good mobile and desktop picross game made using react.js and a small php server to store the results.

The setup of this project is from a previous project.

local you can develop using frankenphp but after build the service can run on almost any php host.

it should be possible to play the game without registering a user account.
